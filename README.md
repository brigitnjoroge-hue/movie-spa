#  Movie Finder SPA

A simple and interactive Single Page Application (SPA) that allows users to search for movies and view details instantly using a public movie API.

---

##  Project Description

Movie Finder SPA is a web application built using **HTML, CSS, and JavaScript**.  
It allows users to search for movies and view results dynamically without refreshing the page.

The app connects to the **The Movie Database (TMDB) API** to fetch real-time movie data such as:
- Movie titles
- Posters
- Ratings
- Release dates
- Descriptions

---

##  Features

- Search for movies instantly  
-  Display movie posters and details  
-  Show ratings and release dates  
-  Modal popup for movie details  
-  Dark mode toggle  
- Save favorite movies (local storage)  
-  Loading spinner for better UX  
- Fully dynamic SPA (no page reloads)

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API / Async-Await
- Local Storage
- TMDB API

---
